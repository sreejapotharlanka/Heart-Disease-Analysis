📖 Overview

Heart disease remains one of the leading causes of death worldwide. Understanding the relationships between lifestyle, demographics, and chronic health conditions can support early awareness and informed decision-making.
This project demonstrates a complete end-to-end analytics pipeline, beginning with SQL-based data preparation, followed by interactive visualization in Tableau, and concluding with deployment through a responsive web application.
Users can interact with the dashboard to explore cardiovascular risk patterns in real time without needing Tableau Desktop.

📚 Tableau Story

The Tableau Story presents the analysis as a step-by-step narrative:

Dataset Overview

Demographic Analysis

Lifestyle Factors

Chronic Health Conditions

Combined Risk Factors

Final Conclusions

This storytelling approach enables users to understand how multiple factors contribute to Heart disease risk.
